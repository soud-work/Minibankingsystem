# Mini Banking System (C++)

A simple console-based banking system written in C++.  
It allows users to:
> Check account balance
> Deposit money
> Withdraw money
> Exit the system safely

 ⚙️ How to Run
1. Copy the `minibank.cpp` file to your computer or Termux.
2. Compile using g++:
   ```bash
   g++ bank.cpp -o bank
3. run it
   ./bank
